# Tache Angular Js -


## Pour Commencer

Pour vous aider à démarrer, vous pouvez simplement cloner le projet et installer les dépendances.

### Installer les dépendances

```cmd
npm install
```


```cmd
bower install```

### Exécuter l'application


```cmd

node server.js

** ou ** 

npm start

```

Maintenant, ouvrir l'application avec : `http://localhost:8000`.
